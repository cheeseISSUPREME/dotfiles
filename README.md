# Kavin's .dotfiles

## Setup
1. `curl -s https://raw.githubusercontent.com/kavin25/.dotfiles/setup.sh | sh`
2. Change Airtable URL CLI env vars - `cd ~/.dotfiles cp url-env-example.sh url-env.sh`
